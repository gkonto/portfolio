IncLens

A terminal-based C++ include inspector that reads a preprocessed .ii file and visualizes include relationships as either a navigable tree or a flamegraph.

view source

stack: Go / Bubble Tea / C++ include analysis

Overview

IncLens is built around a practical workflow: generate a preprocessed C++ translation unit with g++, then inspect the resulting include structure from the terminal.

The point is to make dependency structure easier to understand, surface heavy includes, and give a faster path to compile-time investigation than reading raw preprocessor output.

Top-down tree view

  • Navigate the include hierarchy as a tree.
  • Expand and collapse nodes with keyboard controls.
  • Search includes and jump through matches.
  • Show or hide columns and sort by bytes or lines of code.

Flamegraph view

  • Visualize total lines of code proportionally.
  • Move through the graph with arrow keys.
  • Zoom in with Enter and back out with Esc.
  • Inspect the current file name and line count in the info panel.

Workflow

  • Generate a preprocessed .ii file with g++.
  • Run IncLens against that file.
  • Switch between the tree view and flamegraph depending on whether you want structure or proportional weight.
g++ -E -o example.ii example.cpp
./inclensapp -file example.ii

Why I like it

  • It takes a noisy artifact and turns it into something readable.
  • It stays in the terminal, which fits naturally into a C++ workflow.
  • It focuses on a narrow problem and gives two useful views of the same data.
